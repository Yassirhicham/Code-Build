Radar de Detección - Proyecto Arduino

Introducción

Este proyecto consiste en la implementación de un radar de detección utilizando un sensor ultrasónico, un servo motor y una placa Arduino. La idea es escanear un área determinada midiendo distancias a través del sensor y representarlas visualmente.

Implementación

El sistema se basa en el uso de un sensor ultrasónico HC-SR04, el cual mide la distancia de los objetos cercanos. Este sensor está montado sobre un servo motor, que se encarga de rotarlo en un rango de 15° a 165° para cubrir un amplio campo de visión. Los datos de distancia se envían a través del puerto serie de la placa Arduino, permitiendo su visualización en un software como Processing.

Resultados

El sistema es capaz de detectar objetos a diferentes distancias y representar sus posiciones en tiempo real. Se pueden generar alertas cuando un objeto está a una distancia crítica. Este proyecto tiene aplicaciones en robótica, seguridad y monitoreo de áreas restringidas.

Código Arduino

// Incluye la librería del Servo
#include 

const int trigPin = 10;
const int echoPin = 11;
long duration;
int distance;
Servo myServo;

void setup() {
    pinMode(trigPin, OUTPUT);
    pinMode(echoPin, INPUT);
    Serial.begin(9600);
    myServo.attach(12);
}

void loop() {
    for(int i = 15; i <= 165; i++) {  
        myServo.write(i);
        delay(30);
        distance = calculateDistance();
        Serial.print(i);
        Serial.print(",");
        Serial.print(distance);
        Serial.print(".");
    }
    for(int i = 165; i > 15; i--) {  
        myServo.write(i);
        delay(30);
        distance = calculateDistance();
        Serial.print(i);
        Serial.print(",");
        Serial.print(distance);
        Serial.print(".");
    }
}

int calculateDistance() { 
    digitalWrite(trigPin, LOW); 
    delayMicroseconds(2);
    digitalWrite(trigPin, HIGH); 
    delayMicroseconds(10);
    digitalWrite(trigPin, LOW);
    duration = pulseIn(echoPin, HIGH);
    distance = duration * 0.034 / 2;
    return distance;
}
